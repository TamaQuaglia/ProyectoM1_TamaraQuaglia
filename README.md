# 🎨 Random Color Palette Generator

Una aplicación web que genera paletas de colores aleatorias utilizando JavaScript. Permite visualizar colores en formato HEX, copiar códigos al portapapeles y bloquear colores específicos para mantenerlos mientras se generan nuevos.

## 🚀 Características

* Generación aleatoria de paletas de colores.
* Conversión de colores HSL a HEX.
* Copia de códigos HEX con un solo clic.
* Bloqueo de colores individuales.
* Interfaz moderna y responsiva.
* Actualización dinámica del contenido mediante JavaScript.

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎯 Cómo usar

1. Abrir la aplicación en el navegador.
2. Presionar el botón **Generate Palette**.
3. Visualizar los colores generados.
4. Hacer clic sobre un color para copiar su código HEX.
5. Utilizar el botón de bloqueo para mantener un color fijo.
6. Generar una nueva paleta conservando los colores bloqueados.
